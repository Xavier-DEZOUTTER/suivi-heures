Guide Supabase — 5 minutes
1
Va sur supabase.com et crée un compte gratuit.
2
Clique New project, donne un nom (ex : heures-atelier), choisis un mot de passe, valide.
3
Attends ~1 minute que le projet démarre.
4
Menu gauche → Settings → API. Copie Project URL et la clé anon public, colle-les dans la configuration de l'appli.
5
Menu gauche → SQL Editor. Colle le code ci-dessous puis clique Run :
CREATE TABLE IF NOT EXISTS affaires ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, numero text NOT NULL UNIQUE, libelle text, created_at timestamptz DEFAULT now() ); CREATE TABLE IF NOT EXISTS fabricants ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, prenom text NOT NULL, actif boolean DEFAULT true ); CREATE TABLE IF NOT EXISTS saisies ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, fabricant_id uuid REFERENCES fabricants(id), affaire_id uuid REFERENCES affaires(id), date date NOT NULL, code_activite text, heures numeric(4,1) NOT NULL, created_at timestamptz DEFAULT now() ); ALTER TABLE affaires ENABLE ROW LEVEL SECURITY; ALTER TABLE fabricants ENABLE ROW LEVEL SECURITY; ALTER TABLE saisies ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all" ON affaires FOR ALL USING (true) WITH CHECK (true); CREATE POLICY "allow_all" ON fabricants FOR ALL USING (true) WITH CHECK (true); CREATE POLICY "allow_all" ON saisies FOR ALL USING (true) WITH CHECK (true);
6
C'est prêt ! Reviens sur l'appli, renseigne l'URL et la clé, puis va dans Tableau de bord chef → Gérer pour ajouter tes fabricants et tes affaires.
